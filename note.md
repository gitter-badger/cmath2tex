# CMath2Tex [gh-pages]

# Prerequisites

* `pip install virtualenv`

# Getting Started

```
virtualenv flask
flask/bin/pip install -r requirements.txt
```

## Run local server

* `flask/bin/python run.py`  

## Build static content

* `flask/bin/python freeze.py`  

## Push to github

```
git add --all
git commit --amend
git push -f
```
