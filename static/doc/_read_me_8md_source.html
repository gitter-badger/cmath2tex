1 # CMath2Tex
2 
3 ### Generates `LaTeX` math equations from `C` source code.
4 
5 `CMath2Tex` is the tool to simplify documentation generation for source code with a lot of math expressions.
6 
7 `CMath2Tex` can help you in the following ways:
8 
9 * Quickly generates `LaTeX` equations for huge amount of math expressions in a source code.
10 * Helps with representation of math formula in `LaTeX` format if you don't familiar with syntax.
11 * Generates `Doxygen`-friendly output
12 
13 `CMath2Tex` is developed under macOS, but is set-up to be highly portable. As a result, it runs on Windows and on a variety of Unix flavors as well.

29 # Getting started
30 
31 ```
32 git clone https://github.com/galarius/cmath2tex.git
33 cd cmath2tex
34 mkdir build && cd build
35 cmake -G "Unix Makefiles" ..
36 make
37 ```

55 Here's a quick guide on `pull requests`:
56 
57 1. [Check for open issues](https://github.com/galarius/cmath2tex/issues), or

59  Opening a separate issue to discuss the change is less important for smaller
60  changes, as the discussion can be done in the pull request.
61 2. [Fork](https://github.com/galarius/cmath2tex.git) this repository on GitHub, and start making your changes.
62 3. Check out the README for information about the project setup and usage.
63 3. Push the change (it's recommended to use a separate branch for your feature).
64 4. Open a pull request.
65 5. I will try to merge and deploy changes as soon as possible, or at least leave
66  some feedback, but if you haven't heard back from me after a couple of days,
67  feel free to leave a comment on the pull request.

74 * Operation tree to perform transformations from `C` to `LaTeX`.
